# PetMalu-Information-System
RFID-enabled Application-Based Information System for PetMalu Animal Clinic that streamlines pet patient record management. Instantly retrieves pet profiles, vaccination, deworming, surgery, and visit history through RFID scanning, improving accuracy, security, and clinic workflow efficiency.

🐾 **Application-Based Information System for Pet Patients of PetMalu Animal Clinic Using RFID**

This project is an Application-Based Information System designed for PetMalu Animal Clinic that integrates RFID technology for fast and reliable pet identification.

Each pet is assigned a unique RFID tag linked to its digital medical record. Upon scanning, the system automatically retrieves:

Pet profile information

Owner details

Vaccination records

Deworming history

Surgery history

Visit logs

The system reduces manual record searching, minimizes errors, and improves overall clinic efficiency.
